```yaml
  name: example-argocd
  labels:
    example: basic
spec: {}
```

```bash

kubectl get pods
```

### Expose ArgoCD Service
```bash

```

### Access ArgoCD Dashboard
- Get the admin password:
  ```bash
  kubectl get secret
  kubectl edit secret example-argocd-cluster
  echo <base64-password> | base64 -d
  ```
- Username: `admin`  
- Password: `<decoded-password>`
